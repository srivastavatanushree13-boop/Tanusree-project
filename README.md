Python CLI Calculator with History Tracking
This is a simple command-line calculator built in Python that performs basic arithmetic operations and keeps a history of all calculations performed during the session.
Features
 Supports addition, subtraction, multiplication, and division
Tracks and displays calculation history
 Handles division by zero gracefully
 Input validation for non-numeric entries
 Runs in a loop until the user chooses to exit

 How It Works

The calculator presents a menu of options to the user:
- Add
- Subtract
- Multiply
- Divide
- Show History
- Exit
Each operation logs the result in a session-based history list, which can be reviewed anytime by selecting option 5.
